# daily 1.0.0 功能列表

## 🚧 文件 `file`

### 介绍

管理工作区下的文件

### 功能

1. 文件监听功能
    当工作区下文件有改动时更新工作区域
2. 查询文件树
    列出当前工作区下文件树
2. 新增文件/文件夹
    当前工作区下指定目录下新增文件/文件夹
3. 删除文件/文件夹
    当前工作区下指定目录下删除文件/文件夹
4. 文件/文件夹重命名
    当前工作区下文件/文件夹重命名
    
## 🚧 笔记 `note`

### 介绍

管理笔记

### 功能

1. 基于markdown文件的笔记管理
2. 支持双链
3. 支持笔记回顾
4. 支持日记

## 🚧 计划 `todo`

### 介绍

管理计划

### 功能

1. 支持todo
2. 支持日程
3. 支持看板

## 🚧 记账 `bookkeeping`

### 介绍

管理记账

### 功能

1. 基于账本维度的记账管理
2. 支持标签

## 🚧 其他 

### 可插拔
